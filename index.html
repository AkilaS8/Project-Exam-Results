Features

Responsive Design

This application design responsive to any kind of android device screen size.

Support to the lowest and the latest versions

This application can install the lowest Android version API level 19 (Andriod 4.4 Kitkat version) to the latest API levels.

Awesome Icons

This app includes awesome icons that increase user interaction. These icons help to identify pass, repeat and not attended subjects separately.

GPA Variation

This app includes an animated graph that shows GPA semester-wise. It helps to identify student's progress.

One-time Login

For this app user wants to log in one-time. That time data retrieve from an online database and store local SQLite database. After login user does not need the internet to use this app.

Categorized Results

Users can see results semester-wise, passed subjects, repeat subjects, and not attended subjects separately. If some list is empty for example, the user doesn't have any repeat subjects then it shows an empty message.

Exam Results Publishing System

Exam Results Publishing System is a mobile app. This system connects to the online database. Then get specific student’s results and show them via in this system. Students can view their results by login into this system using Index no. and Registration no. Also, students can view their past subjects, repeat subjects, not attended subjects separately using this system. After students log in to the system, they can view their current GPA and rank. Using this system students can view their all results. Once they log in to the system no need to log in again and again and also no need internet connection again.

This app create for Department of Computer Science Students in Trincomalee Campus, EUSL. So, this system may be helping the students’ lot for tracking their results and keep their GPA marks. And also, help make a path to getting a class.
